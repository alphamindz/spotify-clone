# 🎵 Spotify Clone - Web Player

A fully responsive Spotify Web Player clone built using **HTML5** and **CSS3**. This project uses **Flexbox** and **CSS Grid** to recreate the premium music streaming experience.

## ✨ Features
- **Responsive Layout:** Works on desktops and adjusts for smaller screens.
- **Modern Sidebar:** Functional-looking navigation and library sections.
- **Fixed Player Bar:** A bottom player bar with custom-styled playback controls and progress bar.
- **Sticky Navigation:** The top header stays fixed while scrolling.

## 🛠️ Built With
- **HTML5:** For semantic structure.
- **CSS3:** For styling, including custom scrollbars and grid systems.
- **FontAwesome:** For high-quality icons.

## 📂 Project Structure
- `index.html`: Main structure.
- `style.css`: All styling rules.
- `image/`: Folder for icons and album art.
